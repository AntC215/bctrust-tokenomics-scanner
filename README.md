# BCTrust — Blockchain Project & Tokenomics Scanner

## Project Summary

BCTrust is a blockchain analytics platform concept similar to DexScreener or Birdeye, focused on scanning tokens, tokenomics, project risk, liquidity, and trust indicators.

The project is designed to help users evaluate Web3 assets before interacting with or investing in them.

---

## Problem Statement

Crypto users often need to research tokens across multiple platforms before deciding whether a project is trustworthy.

Important information is scattered across block explorers, token dashboards, liquidity trackers, contract pages, and market data platforms.

This creates risk because users may miss important warning signs such as poor liquidity, concentrated holder distribution, suspicious contract permissions, or unclear tokenomics.

---

## Solution

BCTrust combines blockchain data, market data, liquidity analysis, holder distribution, tokenomics review, and contract risk indicators into one trust-focused dashboard.

## User Stories

### Crypto Researcher

As a crypto researcher, I want to search a token by contract address so that I can quickly review project risk.

### Trader

As a trader, I want to review liquidity and volume so that I can avoid tokens with weak market depth.

### Web3 Investor

As an investor, I want to review holder distribution so that I can understand whether supply is concentrated in a small number of wallets.

### Risk Analyst

As a blockchain risk analyst, I want contract verification and ownership indicators so that I can identify potential red flags.

### New Crypto User

As a newer crypto user, I want a simple trust score so that I can understand whether a token needs deeper research.

## Trust Score Concept

| Category | Purpose |
|---|---|
| Liquidity Health | Reviews available liquidity and pool depth |
| Holder Distribution | Checks whether token supply is concentrated |
| Contract Verification | Reviews whether the token contract is verified |
| Ownership Review | Checks whether ownership is renounced or controlled |
| Market Activity | Reviews price movement, volume, and trading activity |
| Tokenomics Quality | Reviews supply, distribution, and project sustainability |
| Risk Flags | Tracks honeypot, rug-pull, or scam indicators |
