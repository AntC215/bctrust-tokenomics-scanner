# BCTrust User Stories

## Crypto Researcher

As a crypto researcher, I want to search a token by contract address so that I can quickly review project risk.

## Trader

As a trader, I want to review liquidity and volume so that I can avoid tokens with weak market depth.

## Web3 Investor

As an investor, I want to review holder distribution so that I can understand whether supply is concentrated in a small number of wallets.

## Risk Analyst

As a blockchain risk analyst, I want contract verification and ownership indicators so that I can identify potential red flags.

## New Crypto User

As a newer crypto user, I want a simple trust score so that I can understand whether a token needs deeper research.
